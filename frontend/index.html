Case 01 · Option ii · Manual allowance limits

Set allowance limits

Each threshold-based category is scored as (denominator ÷ threshold) ÷ incident count × weight, capped at its own weight. Zero incidents in a period awards the full weight. Category weights are fixed by the formula sheet and aren't editable here — only the thresholds are.

Ratio-based categories aren't threshold-driven, so nothing to set here: OGRA Fleet %, DDT %, Medical Screening %.
